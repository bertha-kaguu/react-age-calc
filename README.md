# Elite Age Calculator (React)

A modern Age Calculator Web Application built with React that calculates a person's age in years, months, days, and seconds.
The app also shows the user's zodiac sign, next birthday countdown, and includes a progress bar showing the time remaining until the next birthday.

This project demonstrates modern frontend development techniques including component-based architecture, state management, and persistent data storage.

---

## Features

* Calculate age in years, months, and days
* Live age in seconds counter
* Next birthday countdown
* Zodiac sign detection
* Birthday progress bar
* Dark mode toggle
* Multiple people support
* LocalStorage data persistence
* Responsive modern UI
* Built using reusable React components

---

## Technologies Used

* React – Frontend library
* JavaScript (ES6+)
* CSS3
* HTML5
* LocalStorage API
